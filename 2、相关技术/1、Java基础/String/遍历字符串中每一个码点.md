## 1、遍历一个字符串，打印出其中每一个码点

```java
String s = "我🐂🐍😊";
for(int i = 0;i < s.length();){
    int value = s.codePointAt(i);
    if(Character.isSupplementaryCodePoint(value)){
        System.out.println(s.substring(i,(i+2)));
        i+=2;
    }else{
        System.out.println(s.substring(i,(i+1)));
        i++;
    }
}
```

## 2、相关API

1. boolean equals(Object o)

2. int hashCode()

3. int length()：返回代码单元的数量（字符数组的长度）

4. <font color = "red">int codePointCount(0,str.length())：</font>返回码点的数量（字符串长度）

5. <font color = "red">char charAt(int index)：</font>返回指定位置的代码单元

6. <font color = "red">String substring(int beginIndex,int endIndex)：</font>返回一个新的字符串，它是此序列的一个子字符串（能获取多个代码单元）

7. <font color = "red">int codePointAt(int index) ：</font>返回指定索引处的字符对应的码点（整数值）

8. int indexOf(int ch)：返回指定字符在此字符串中第一次出现的索引

9. String intern()：当调用 intern 方法时，如果池已经包含一个等于此 String 对象的字符串（用 equals(Object) 方法确定），则返回池中的字符串。否则，将此 String 对象添加到池中，并返回此 String 对象的引用。

   返回：一个字符串，内容与此字符串相同，但一定取自具有唯一字符串的池

10. int offsetByCodePoints(int index, int codePointOffset)  ：返回此 `String` 中从给定的 `index` 处偏移  `codePointOffset` 个代码点的索引

11. String trim()：返回字符串的副本，忽略前后的空白

12. String[] split(String regex)：根据给定的正则表达式来拆分此字符串

13. String valueOf(int i)：返回int参数的字符串表示形式
