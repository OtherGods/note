# 本地运行

### 1、安装`node_modules`:

```
npm install
```

> 如果提示npm 不可用，则可以执行：`brew install node` （mac 系统），其他系统可以百度下 node 的安装方式

如果安装后，npm install 执行一直转圈圈，可以换以下方式：

```
npm install -g yarn
yarn install
```

### 2、修改后端服务地址：

找到proxy.ts 文件，修改其中的代理地址（之所以用代理是为了解决跨域问题），假如你的nft-turbo-admin 的地址是：`http://localhost:9001`

那么按照以下修改：
```shell
export default {  
	// 如果需要自定义本地开发服务器  请取消注释按需调整  
	dev: {  
	       // localhost:8000/api/** -> https://preview.pro.ant.design/api/**  
	  '/admin/': {  
	               // 要代理的地址  
	    target: 'http://localhost:9001',  
	               // 配置了这个可以从 http 代理到 https  
	               // 依赖 origin 的功能可能需要这个，比如 cookie  
	    changeOrigin: true,  
	  },  
	}  
}
```

### 3、项目启动

使用命令`npm start` 启动前端应用，启动后，就可以通过`http://localhost:8000/user/login` 访问页面了：

# 服务器部署

### 1、安装 Nginx

参考这篇文章，把nginx 给安装并启动好：https://blog.gaomeluo.com/archives/UbuntuNginx/

### 2、前端代码获取

在前端代码目录下，执行`npm run build` ，就会编译前端代码，都会编译到 dist目录下：
![image.png](https://raw.githubusercontent.com/OtherGods/MaterialImage/main/img/202411042334649.png)

然后把这些代码都复制到你部署了 nginx 的服务器上。把他们放在一个固定的目录下，比如`/dist/` 。

### 3、修改 nginx 配置

修改你的 nginx 配置，在/etc/nginx/nginx.conf

增加如下配置：
```nginx
http {  
  server {  
	  listen       80;  
	  server_name  localhost;  
	  
	  # 这里是添加的内容  
	  location / {  
		  root /dist/;  
		  index index.html;  
	  }  
  }  
}
```

在你的 http 标签下 ，把 server 添加进去，端口号也可以改成你自己想用的，总之就是目录跳转到我们的前端代码存放目录即可。这里的 localhost 改成你自己的IP地址或者域名。

然后就可以用你的 ip 地址+端口访问了：
