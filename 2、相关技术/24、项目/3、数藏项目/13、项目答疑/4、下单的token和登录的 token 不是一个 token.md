token是一个通用概念，一般表示口令。或者叫做一个凭证，广泛的应用在登录、鉴权等领域中。

我们的项目中有多个地方用到了token，有的是我们自己主动用的，如下单的时候使用token机制来判断是否重复下单。

也有的是框架中自带的token，如sa-token进行登录鉴权的时候他也会有一个自己的token来识别用户的登录信息。

但是，他们绝对不是同一个token。此 token 非彼 token。登录的 token 是 satoken，只用于登录鉴权。
