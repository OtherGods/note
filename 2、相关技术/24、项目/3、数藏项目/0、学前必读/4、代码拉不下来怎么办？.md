如果你已经获得了代码权限，压缩包你能下载，但是通过git命令拉不到代码。那么说明你的配置是不正确的，

那么就需要通过以下2种方式之一进行配置。

### 通过访问令牌

在你的个人设置中，创建访问令牌
![image.png](https://raw.githubusercontent.com/OtherGods/MaterialImage/main/img/202411032335611.png)

输入名称，并且勾选上对应的权限，创建即可
![image.png](https://raw.githubusercontent.com/OtherGods/MaterialImage/main/img/202411032336414.png)

创建成功后，他会提示你令牌内容，这个一定要记好。
![image.png](https://raw.githubusercontent.com/OtherGods/MaterialImage/main/img/202411032336220.png)

我创建的：pt-1TnYV3WGHQThRy3O1B2394OS_033e196d-b5a1-4162-839b-da2de1a113b2

这样你就可以通过以下方式拉取代码：
```git
git clone https://codeup.aliyun.com/66263f57d833774c93cc0418/NFTurbo/NFTurbo_Server.git
正克隆到 'NFTurbo_Server'...
Username for 'https://codeup.aliyun.com':这里输入你的用户名
Password for 'https://xxx@codeup.com':这里输入你的访问令牌
```

就可以把代码拉下来了。

**如果访问令牌忘了咋办？**

没关系，重新再申请一个就行了。

### 通过SSH公钥

如果你不想记这个访问令牌，或者你觉得麻烦，也可以通过SSH的方式来拉代码、
![image.png](https://raw.githubusercontent.com/OtherGods/MaterialImage/main/img/202411032342089.png)

这里创建SSH公钥。

创建方式根据你不同的操作系统参考这个文档操作就行了。https://help.aliyun.com/document_detail/153709.html

操作完之后，你就可以免密进行代码拉取了，但是需要注意这种方式，需要通过SSH协议进行代码拉取，不要用http：
![image.png](https://raw.githubusercontent.com/OtherGods/MaterialImage/main/img/202411032342294.png)
