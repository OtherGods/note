    * 两数之和——利用缓存思想（实际示例：解决循环依赖的三级缓存）

        int[] nums = {2, 7, 11, 15, 3, 6, 5, 4};  
        int target = 9;  
        Map<Integer,Integer> map = new HashMap<>();  
        for(int i = 0; i < nums.length; i++){  
            Integer integer = map.get(target - nums[i]);  
            if(integer != null){  
                return new int[]{i, integer};  
            }  
            map.put(nums[i],i);  
        }  
        return new int[2];  
    }

        }
