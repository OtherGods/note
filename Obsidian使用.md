# 1、搜索

官网：[搜索](https://publish.obsidian.md/help-zh/%E6%A0%B8%E5%BF%83%E6%8F%92%E4%BB%B6/%E6%90%9C%E7%B4%A2)
搜索示例：
- `path:hollis MVCC`：查询指定的`hollis`路径下包含名称或内容`MVCC`的文件
	- 被空格分隔的内容会被视为一个独立的条件，只有文件满足所有独立条件，文件才会作为搜索结果返回
- 搜索内容包含引号，你需要在引号前添加反斜杠（`\`）来转义，例如用 `他们说\"你好\"` 来搜索`他们说"你好"`

# 快捷键

部分快捷键
1. `Ctrl + Shift + f`：搜索
2. `Ctrl + Alt + s`：设置
3. `Ctrl + k`：插入链接
4. `Ctrl + g`：查看所有文件关系图谱
5. `Ctrl + n`：新建笔记
6. ……
