# 典型回答

301 和 302 都是 HTTP 重定向状态码，所谓重定向就是从一个URL跳转到另一个URL。

但301和302还是有较大的区别的：

**301 是永久重定向**。资源已经永久移动到新的 URL，浏览器和搜索引擎**会缓存这个重定向结果**，适合永久链接更换。

**302 是临时重定向**，资源临时移动到新的 URL，客户端**不应该缓存**，后续请求仍然访问旧 URL。
