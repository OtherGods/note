# 引言

**文末有我的面试复习资料，都是我面试各大厂后，总结的一些考题，之前靠这些收获了百度、滴滴、美团、新浪等大厂 Offer，需要的同学可以私我。**



这里面包括后端技术栈掌握的所有知识，主要是给你搭建一套知识体系框架，让你有一个全局的认识。



因为应届生和工作的同学，面试时考察的知识点不一样，所以我将内容划分为“应届生必学”和“工作必学”，让你学习更优针对性，文末有学习资料，都是我的原创文章。



# 一、知识列表

## **1.** **语言栈知识**

### Java：

1. Java 基础（必学）
2. JVM（必学）
3. 并发编程（必学）
4. Spring 基础（必学）
5. Spring 源码（选学）
6. Spring Boot（应届生选学，工作必学）
7. MyBatis（应届生选学，工作必学）
8. Maven（应届生选学，工作必学）



## **2.** **后端知识**

操作系统（必学）：学好本科书籍即可，后面也会推荐一些学习资料。

计算机网络（必学）：学好本科书籍即可，后面也会推荐一些学习资料。

数据结构（必学）：学好本科书籍即可。

算法（必学）：刷 LeetCode 即可，这个就是需要自己多刷，参考书籍《剑指Offer》



### MySQL：

详见书籍《MySQL技术内幕-InnoDB存储引擎》

1. 索引（必学）
2. 事务（必学）
3. 锁（必学）
4. 常见优化策略（必学）
5. 主从同步（应届生选学，工作必学）



### Redis：

详见书籍《Redis设计与实现》

1. 5 种基础结构实现机制（必学）
2. AOF 和 RDB 日志（必学）
3. 主从同步（应届生选学，工作必学）
4. 哨兵机制（应届生选学，工作必学）
5. 分片集群（应届生选学，工作必学）



### 设计模式：

掌握常用的 7-10 种设计模式即可，后面也会推荐一些学习资料。

1. 工厂模式（必学）
2. 单例模式（必学）
3. 模板模式（必学）
4. 策略模式（必学）
5. 组合模式（必学）
6. 代理模式（必学）
7. 装饰器模式（应届生选学，工作必学）
8. 责任链模式（应届生选学，工作必学）
9. 门面模式（应届生选学，工作必学）
